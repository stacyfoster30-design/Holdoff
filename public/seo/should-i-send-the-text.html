Should I Send the Text?

You already drafted it twice. Here's how to know if you should.

The Real Question Behind the Question

"Should I send this?" is almost never really about the message. It's about what the message is doing. Is it saying something, or is it managing anxiety? Is it communication, or is it a bid for reassurance? That distinction changes the answer.

The Hold-Off Test

  • Why am I sending this right now? If the honest answer is "because the silence is unbearable" — hold off.
  • What does this message need from them? If it needs them to respond in a specific way for you to feel okay — reconsider.
  • Is there new information in here? If not, what is this message actually adding?
  • How will I feel if they don't reply? If the answer is "terrible" — that feeling is already here, and sending the message won't make it go away.

When to Send It

When you have something genuinely new to say. When it doesn't require a specific response to land okay with you. When you'd be comfortable not getting a reply for 24 hours. When you're not in the middle of a spiral.

When to Hold Off

When you're in the feeling. When you've already sent something today that went unanswered. When the honest reason is "I need to know they're still there." When you drafted it more than twice.

Common Questions

Is it okay to send a text if I'm not sure?

Run it through HoldOff. Or wait 20 minutes. If it still feels necessary after 20 minutes, your original instinct was probably right.

When is double texting okay?

When you have something genuinely new to add, or when you legitimately forgot something material. Not when you're following up because the silence is making you anxious.

What if I already sent it and regret it?

Sit with the discomfort. Sending a follow-up to address the previous send usually makes it worse. Most things that feel cringeworthy to the sender are interpreted much more neutrally by the receiver.
